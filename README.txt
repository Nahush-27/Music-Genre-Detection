The folder contains the following:
1. Music_Genre_Classification_2.ipynb - python code for the project
2. Results
  i. shuffle=false.png - accuracy results of different models if shuffle=False during K-Fold Cross Validation
 ii. kfold_accuracy.png - accuracy results of different models (if shuffle=True) using K-Fold Cross Validation
iii. test results.png - Training RandomForestClassifier on train and accuracy on test set
 iv. confusion_matrix.png - confusion matrix for RandomForestClassifier trained on train dataset
  v. newtest.png - prediction result for a particular input feature set
3. CS_419_Project - Project Report
